vibra — Free End-to-End Encrypted Chat App

vibra is a free, open-source, browser-based encrypted chat application. No account, no email, no phone number required. Start a private encrypted chat room in seconds. All messages are encrypted with AES-256-GCM using ECDH P-256 key exchange directly in your browser via the Web Crypto API. The server never sees your private keys or plaintext messages.

Why vibra is the most secure free chat app

Unlike other messaging apps, vibra performs all cryptographic operations client-side. Your private key is generated in your browser and never transmitted to our servers. We use the same class of encryption as Signal, WhatsApp, and iMessage — ECDH key exchange combined with AES-256-GCM authenticated encryption — but without requiring an account or installing an app.

Encryption Technical Details

vibra uses the Web Crypto API built into every modern browser to perform cryptographic operations without any external library dependencies. The key exchange protocol is Elliptic Curve Diffie-Hellman (ECDH) on the NIST P-256 curve. The shared secret is then processed through HKDF with SHA-256 to produce a 256-bit AES-GCM key. Every message is encrypted with a fresh random 96-bit IV. The GCM authentication tag provides integrity protection — any tampered message will fail to decrypt.

Frequently Asked Questions

Is vibra truly end-to-end encrypted?

Yes. vibra uses ECDH P-256 for key exchange, HKDF-SHA-256 for key derivation, and AES-256-GCM for all message encryption. All cryptographic operations run in your browser. The vibra server only relays encrypted bytes and has no mathematical ability to decrypt your messages.

Do I need to create an account to use vibra?

No. vibra requires zero registration. Enter a display name and a room name — that's all. No email, no password, no phone number, no personal data collected.

Are messages stored on the server?

No. vibra is fully ephemeral. No messages are written to disk or any database. Chat rooms exist only in server memory and are permanently destroyed when all users disconnect. A server restart erases all data.

Can I share files securely on vibra?

Yes. vibra supports sharing images (JPEG, PNG, GIF, WebP) with inline preview, audio files (MP3, OGG, WAV, FLAC) with a built-in player, video files (MP4, WebM), PDFs, ZIP archives, Word documents, and plain text files. Maximum file size is 20 MB.

What makes vibra different from other encrypted chat apps?

vibra runs entirely in your browser with no app installation. Unlike Signal or WhatsApp, you don't need a phone number or account. Unlike Telegram, messages are end-to-end encrypted by default and never stored. Unlike Wickr or Keybase, it's completely free and open source. vibra is the easiest way to have a truly private conversation online.

Is vibra open source?

Yes. vibra is fully open source and can be self-hosted on your own server. The server-side code (Node.js + Express + Socket.IO) and client-side code are available for inspection and auditing.

Does vibra work on mobile?

Yes. vibra is fully responsive and works on iOS Safari, Android Chrome, and all modern mobile browsers. It supports safe-area insets for notched phones.

Privacy Policy Summary

vibra collects no personal data. We do not use cookies, analytics, or tracking pixels. We do not serve advertisements. Your IP address may appear in server logs temporarily but is not stored permanently. Files uploaded are stored temporarily and deleted when the server restarts. We do not share any data with third parties.

How to use vibra

  1. Open vibra in any modern web browser — no installation needed
  2. Enter any display name (anonymous, no real name required)
  3. Enter a room name to create or join a chat room
  4. Share the room name with anyone you want to chat with privately
  5. Messages are automatically encrypted before leaving your browser
  6. Use the paperclip button to attach images, audio, video, or documents
  7. Click the shield badge to view encryption details
  8. Click "leave" to exit — the room is destroyed when everyone leaves
